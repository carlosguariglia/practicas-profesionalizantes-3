	
Framework de trabajo (Parte II)
Formato de entrega: Repositorio git en ruta : sdk/v2

Consigna: A partir del código desarrollado en sdk/v1, que incorpora una primera aproximación a la construcción del framework de trabajo para las prácticas y el modelado de datos trabajado en clase (30/4). 

Se solicita:

-Desarrollar un módulo de autenticación y gestión de permisos. (Modelado parcial de base de datos guiado por el docente)

En primer lugar deberán identificar todas las funcionalidades de alta, baja, modificación que correspondan para este caso de uso. 
Incorporar las modificaciones sobre la estructura del trabajo realizado en sdk/v1. Cada funcionalidad tendrá que tener un nombre consistente.

-A fines de poder trabajar con la base de datos de manera adecuada y poder hacer pruebas de querys, solicite a alguna IA, una query de inserción que le permita tener una considerable cantidad de registros en la base de datos para poder experimentar con la misma.


Consideraciones

-No incluya en el repositorio de trabajo la carpeta node_modules
-No modularice la interfaz gráfica en archivos separados (código html) 
-No utilice frameworks
